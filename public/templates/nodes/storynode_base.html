This is a tool to talk technically.
LightningCatcher.org is a place and a tool to study together online. In particular, we provide a platform which supports technical typesetting and display - inlcuding mathematics, source code, and a nade- rather than thread- based discussion architecture. We are built on Django and MathJax and InfoVis.js, and should run well on any modern browser.
